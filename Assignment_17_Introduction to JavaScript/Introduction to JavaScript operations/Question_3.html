Differentiate between unary, binary, and ternary operators in JavaScript. Give examples of each.

Unary Operators Binary Operators Ternary Operators
Unary operators work with a single operand. Binary operators work with two operands. The ternary operator is the only operator in JavaScript that takes three operands.
They can appear before or after the operand. They are typically infix operators, appearing between their operands. It's a conditional operator that evaluates a condition and returns one of two values based on
Examples include unary plus (+), unary minus (-), typeof, ++ (increment), -- (decrement), etc. Examples include addition (+), subtraction (-), multiplication (*), division (/), assignment (=), etc. It's written in the form of condition ? exprIfTrue : exprIfFalse.
