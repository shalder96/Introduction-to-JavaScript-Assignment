Describe the categorization of operators in JavaScript based on their functionality. Provide examples for each category.

In JavaScript, operators can be categorized based on their functionality. Here's a breakdown of these categories with examples for each:

  1. Arithmetic Operator
    • Addition(+):Adds two operands or concatenates strings.
    • Example:let sum = 10 + 5; // sum equals 15 let str = 'Hello ' + 'World'; // str equals 'Hello World'
    • Subtraction (-):Subtracts one operand from another.
    • Example:let difference = 10 - 5; // difference equals 5
    • Multiplication (*):Multiplies two operands.
    • Example:let product = 10 * 5; // product equals 50
    • Division (/):Divides one operand by another.
    • Example:let quotient = 10 / 5; // quotient equals 2
    • Modulus (%):Returns the remainder of the division of one operand by another.
    • Example:let remainder = 10 % 3; // remainder equals 1
  2. Comparison Operator
    • Equal to (==):Compares two operands for equality, but performs type coercion if necessary.
    • Example:console.log(10 == '10'); // true (type coercion)
    • Strict Equal to (===): Compares two operands for equality without type coercion.
    • console.log(10 === '10'); // false (no type coercion)
    • Not Equal to (!=):Compares two operands for inequality, but performs type coercion if necessary.
    • Example:
      • console.log(10 != '10'); // false (type coercion)
    • Strict Not Equal to (!==):Compares two operands for inequality without type coercion.
    • Example:console.log(10 !== '10'); // true (no type coercion)
    • Greater than (>), Greater than or equal to (>=), Less than (<), Less than or equal to (<=): Compare two operands numerically or alphabetically.
  3. Logical Operators
    • Logical AND (&&): Returns true if both operands are true.
    • Logical OR (||): Returns true if at least one operand is true.
    • Logical NOT (!): Returns true if the operand is false and vice versa.
  4. Assignment Operator
    • Assignment (=):Assigns a value to a variable.
    • Example:let x = 10;
    • Compound Assignment (+=, -=, *=, /=, %=): Performs an arithmetic operation and assigns the result to the variable.
    • Example:let y = 5; y += 2; // Equivalent to y = y + 2; (y equals 7)
  5. Assignment Operator
    • Assignment (=):Assigns a value to a variable.
    • Example:let x = 10;
    • Compound Assignment (+=, -=, *=, /=, %=): Performs an arithmetic operation and assigns the result to the variable.
    • Example:let y = 5; y += 2; // Equivalent to y = y + 2; (y equals 7)
  6. Unary Operators
    • Unary Plus (+):Converts an operand into a number.
    • Example:let num = +'10'; // num equals 10
    • Unary Minus (-)Converts an operand into a negative number.
    • Example:let negativeNum = -10; // negativeNum equals -10
  7. Bitwise Operator:
    • Bitwise AND (&), Bitwise OR (|), Bitwise XOR (^), Bitwise NOT (~):Perform bitwise operations on operands.
    • Example:let result = 10 & 5; // result equals 0 (bitwise AND)
  8. Conditional (Ternary) Operator:
    • Conditional Operator (?:):Evaluates a condition and returns one of two values based on whether the condition is true or false.
    • Example:let age = 18; let message = (age >= 18) ? 'Adult' : 'Minor'; // message equals 'Adult'
